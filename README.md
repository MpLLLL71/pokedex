# PokéOnline – Pokédex Pokémon Interactif

Un Pokédex moderne et stylé inspiré de l'univers Pokémon, avec un design rétro pixel-art et des animations fluides.

## Fonctionnalités principales

- Navigation par **générations** (Gen 1 à Gen 9 + mode "All")
- Affichage paginé des Pokémon (10 / 20 / 50 / Tous)
- Recherche intelligente par **ID**, **nom** ou **type**
- Cartes Pokémon avec **gradient de type** (mono ou double-type)
- Modal détaillé avec :
  - Artwork officiel
  - Infos hauteur/poids
  - Types stylisés
  - **Graphique radar** des stats (via Chart.js)
- Design rétro avec police **Press Start 2P**, ombres noires, gradients rouge/jaune/noir

## Technologies utilisées

- HTML5 / CSS3
- JavaScript (ES6+)
- [PokeAPI](https://pokeapi.co/) (données des Pokémon)
- [Bootstrap 5.3](https://getbootstrap.com/) (modal, alertes, tabs)
- [Chart.js](https://www.chartjs.org/) (radar stats)
- Google Fonts – Press Start 2P
- LocalStorage (simulation connexion)
